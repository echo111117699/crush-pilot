# Crush Pilot

> 不看星盘、不算塔罗——基于真实对话数据的理性关系分析工具。

Crush Pilot 帮助年轻女性在暧昧关系中保持清醒、获得主体性。上传聊天截图，AI 解读双方行为模式，积累动态人格洞察，并在关键决策点提供高价值策略。

## 线上体验

👉 **[echo111117699.github.io/crush-pilot](https://echo111117699.github.io/crush-pilot/)**

## 核心功能

| 功能 | 说明 |
|---|---|
| **AI 聊天分析** | 上传聊天截图，获取护盾语 + 客观分析 + 高价值策略 |
| **策略滑块** | 三段式选择（试探 ↔ 适中 ↔ 直接），含淡入淡出动画 |
| **人格洞察积累** | 每次分析提取行为证据，持续积累"关于他"和"关于你"的双线档案 |
| **静默期工具** | SVG 圆形倒计时 + 替代活动建议，帮用户避免冲动联系 |
| **每周高光** | 关系模式发现与情绪趋势回顾 |

## 产品设计

- **"他"线**：上传截图 → AI 分析 → 行为模式提取 → 人格洞察积累 → 策略建议
- **"我"线**：自我档案 → 使用数据 → 关系模式发现 → 静默期 → 每周高光

更多产品设计思路见 [草图作业文档](docs/草图作业.md)。
